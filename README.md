# Ski Master

A top-down vertical scroller game about Skiing made using [Flame Engine](https://flame-engine.org/)

![Demo](branding/SkiMaster-Demo.gif)

## Download/Play

Coming soon

## Build/Run Steps

```bash
# Clone this project
$ git clone https://github.com/ufrshubham/ski_master

# Access
$ cd ski_master

# Install dependencies
$ flutter pub get

# Run the project
$ flutter run
```
